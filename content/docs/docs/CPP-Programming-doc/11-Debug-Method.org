* GDB
** 调试符号
在编译的时候加上 ~-g~ 选项，以使程序能够执行调试。如果是 ~makefile~ ，可以指定全局选项。

#+begin_src makefile
  CXX = g++
  CXXFLAGS = -Wall -g
#+end_src

** 调试界面
调试界面包括两种：
1. ~gdb~ 不显示程序代码界面，仅回显步进执行结果
2. ~gdb --tui~ 提供执行的字符界面
